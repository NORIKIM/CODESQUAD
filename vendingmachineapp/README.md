## 자판기 아이패드 앱

**요구사항**

- 자판기의 기능과 화면을 아이패드용으로 구현
  - 자판기의 음료 추가
  - 음료 구매 시 해당 음료 재고 삭감
  - 음료 구매 시 해당 음료 가격 만큼 잔액 소진
  - 음료를 구매 할 수 있도록 잔액 증가
- 화면에는 버튼,이미지뷰,레이블이 쓰임

**프로젝트를 마치고**

앱을 처음 만들어 보게 되면서 알아야 하는 라이브러리, 프레임워크가 많아져서 답답하고 힘들었지만 애플 문서와 구글링을 통하여 문제를 해결해 나갔다.

컨트롤러와 딜리게이트를 집중적으로 학습
